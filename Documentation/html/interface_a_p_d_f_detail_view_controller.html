Detailed Description

The detail view controller of the split view in this iPad app. It displays the PDF file.

Is performed by the action button in the navigation bar and toggels if the new annotations popover should appear or disappear. If the popover is already visible the popover gets dismissed else the new annotations view controller gets shown
