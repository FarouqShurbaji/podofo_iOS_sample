Detailed Description

A class that models an annotation with a given annotation type, and parameters that specify the annotation. For FreeText Annotatons the font type, the text and the text color are specified. For Square and Highlight annotations the property a annotation color is specified instead of the text color. All Annotations hold a view that has the right size and contains the annotation. This view can be drawn into the PDF.
